# The ultimate keymap for macOS

> 这只是一个macOS app 快捷键的统一规范，希望开发者们都按照此规范，那么世界就清净了。

* `cmd+c` copy
* `cmd+v` paste
* `cmd+x` cut
* `cmd+z` undo
* `cmd+a` select all
* `cmd+f` find
* `cmd+o` open
* `cmd+n` new
* `cmd+s` save
* `cmd+w` close
* `cmd+tab` switch apps
* `cmd+~` switch app's windows
* `cmd+q` quit
* `cmd+,` preferences
